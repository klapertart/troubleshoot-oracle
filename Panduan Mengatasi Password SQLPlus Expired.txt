Panduan mengatasi password EXPIRED pada SQLPLUS


1. Masuk ke window server yang ada di virtualbox

2. Masuk ke SQL Plus 8.0 >> (Klik Start > All Programs > Oracle for Windows NT > SQL Plus 8.0)

3. Akan muncul dialog box login, masukan User Name: system, Password: sismiop, Host String: sismiop, lalu tekan OK

4. Akan muncul keterangan password expired, isi old password: sismiop, New Password: sismiop, Retype New Password: sismiop, lalu tekan OK
